# ⚽ Futsal Expense Manager

> Split turf fees with your squad — no arguments, no confusion.  
> A lightweight, mobile-first web app that tracks who paid and who still owes.

---

## ✨ Key Features

- **Add Players Dynamically** — Type a name and hit Add Player. Duplicate names are automatically rejected.
- **Live Fee Calculation** — Enter the total turf cost and the app instantly calculates each player's exact share.
- **Paid / Due Toggle** — One tap switches a player between Paid ✓ and Due status with color-coded feedback (green / red).
- **Real-time Summary** — Collected and Due amounts update instantly as you toggle players.
- **LocalStorage Integration** — Player list, fee amount, and payment statuses are saved to the browser. Refresh the page — everything is still there.
- **Session Reset** — One click clears everything and starts a fresh session.
- **Mobile-First UI** — Built for the football ground. Works perfectly on any phone screen.

## 🚀 How to Run Locally

1. Clone the repository:
```bash
   git clone https://github.com/babluverse/futsal_expense_manager.git
```

2. Open the project folder:
```bash
   cd futsal_expense_manager
```

3. Open `index.html` directly in your browser — no setup, no dependencies needed.
